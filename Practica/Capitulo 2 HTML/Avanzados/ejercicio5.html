Introducción

La API Ficticia es una herramienta diseñada para facilitar la integración de servicios en aplicaciones web. Esta API permite realizar solicitudes HTTP para acceder a información en tiempo real sobre diversos servicios. A continuación, se presenta la documentación necesaria para su implementación.

Instalación

Para comenzar a usar la API Ficticia, primero necesitas instalarla. Puedes hacerlo de la siguiente manera:

npm install api-ficticia

Una vez instalada, puedes importar la API en tu proyecto de la siguiente forma:

const api = require('api-ficticia');

Uso

Para utilizar la API Ficticia, debes realizar una solicitud a su endpoint principal:

GET https://api.ficticia.com/datos

Asegúrate de incluir tu clave de API para autenticarte. Aquí tienes un ejemplo de cómo hacerlo:

api.get('/datos', { headers: { 'Authorization': 'Bearer tu_clave_api' } })
    .then(response => console.log(response.data))
    .catch(error => console.error(error));

Ejemplos

Ejemplo de cómo hacer una solicitud GET a la API para obtener datos:

const axios = require('axios');
axios.get('https://api.ficticia.com/datos', {
    headers: {
        'Authorization': 'Bearer tu_clave_api'
    }
})
.then(response => {
    console.log(response.data);
})
.catch(error => {
    console.error('Error:', error);
});

Otro ejemplo utilizando fetch:

fetch('https://api.ficticia.com/datos', {
    method: 'GET',
    headers: {
        'Authorization': 'Bearer tu_clave_api'
    }
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
